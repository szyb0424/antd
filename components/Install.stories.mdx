## 安装
使用 npm 或 yarn 安装
```sh
npm install @yub04/antd --save
```
```sh
yarn add @yub04/antd
```
## 浏览器引入
在浏览器中使用 script 和 link 标签直接引入文件，并使用全局变量 `antd`
我们在 npm 发布包内的 `@yub04//dist` 目录下提供了 `ant.js`

## 示例

```js
import { Button } from '@yub04/antd';
ReactDOM.render(<Button>按钮</Button>, mountNode);
```
